  ## Usage

  ### Setup Instructions

  1. Clone the repo

  ```shell
  git@github.com:fondofhats/employee-tracker.git
  ```

  2. Move to the project root directory

```shell
cd employee-tracker
```

3. Run npm install

```shell
 npm install
 npm install mysql2
 npm install inquirer
 npm install console.table
 npm install figlet

```
